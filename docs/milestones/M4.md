# M4：Windows 支持

**目标**：Windows 用户双击 .exe 安装安装器，通过图形界面完成 WSL2 + Ubuntu + OpenClaw 全流程安装。

**验收标准**：
- [ ] 检测 WSL2 状态（已有/需安装/不支持）
- [ ] 引导安装 WSL2（需要 UAC 提权，安装后提示重启）
- [ ] 重启后继续：在 WSL Ubuntu 中执行 install-ubuntu.sh
- [ ] 备选：原生 Windows 路径（PowerShell 安装 Node.js + OpenClaw）
- [ ] NSSM 注册 Windows 服务（开机自启）
- [ ] .msi 安装包可在 Windows 10/11 x64 正常安装

## Windows 安装流程图

```
启动安装器
    ↓
检测 WSL2
    ├── WSL2 可用 → 在 WSL Ubuntu 中执行 install-ubuntu.sh
    ├── WSL2 未安装 → 引导安装（需重启）→ 重启后继续
    └── 不支持 WSL → 原生 PowerShell 路径
```
